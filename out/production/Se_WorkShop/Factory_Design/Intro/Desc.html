Creational Pattern

Factory Handles the details of an object

Define interface to create an object


Used in frameworks : at the abstract level

Framework uses the abstract classes to define the relationship between objet

Use factory method when

  1. a class can't anticipate the class of object it must create
  2. a class want its subclass to specify the object it creates
  3. when implementing the parallel class hierarchy : no of responsibilities shift from one class to another class
